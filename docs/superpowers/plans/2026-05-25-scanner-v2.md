# Scanner V2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the fragmented photo/barcode flow with a full-screen in-app scanner that feels native and premium.

**Architecture:** Add a single `ScannerScreen` powered by `expo-camera` for live meal capture, barcode detection, nutrition-label capture, and gallery entry. Keep product lookup and AI analysis in `App.tsx`; the scanner only captures or emits scan events.

**Tech Stack:** React Native, Expo SDK 54, `expo-camera`, `expo-image-picker`, Vitest.
